# Instalando R en linux desde las fuentes

Esta guia permite hacer la instalacion de R para desarrollo de paquetes. Asumiremos que disponemos de un sistema Linux basado en Debian (como Ubuntu y sus derivados). Un enfoque alternativo es instalar `r-base-dev` usando, por ejemplo, el [Gestor de paquetes Synaptic](https://github.com/mvo5/synaptic)
